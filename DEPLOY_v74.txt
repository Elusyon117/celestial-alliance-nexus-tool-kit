CELESTIAL NEXUS v74 — WIKELO ARTWORK + PERFORMANCE UPDATE
==========================================================

WHAT CHANGED
------------
• Added bundled artwork for 10 previously blank Wikelo Trade Center entries.
• Exact Wikelo table artwork is used where it is publicly available.
• Polaris Bit, Killshot Dominion, R97 Crimson and Monde Crimson use clearly labeled visual references because dedicated exact-item artwork is not currently available from the referenced wiki pages.
• Wikelo images now load from this repository instead of repeatedly fetching those 10 images at runtime.
• Remaining remote artwork loads only near the viewport, with a maximum of three lookups at once.
• Removed the previous multi-query / multi-probe process that could make cards wait a long time.
• Updated the artwork cache key and the PWA service-worker cache to v74.

BROWSER-ONLY GITHUB DEPLOYMENT
------------------------------
1. Extract Celestial_Nexus_v74_GitHub_Pages_Deploy.zip on your computer.
2. In GitHub, open:
   Elusyon117/celestial-alliance-nexus-tool-kit
3. Confirm the branch selector says main.
4. Choose Add file → Upload files.
5. Drag the CONTENTS of the extracted package onto the upload page.
   Do not upload the ZIP itself.
6. Confirm the upload includes at least:
   index.html
   sw.js
   assets/wikelo/boomtube-clanguard.webp
   assets/wikelo/palatino-mark-1.webp
   assets/wikelo/geist-snow.webp
   assets/wikelo/ana-endro.webp
   assets/wikelo/bokto.webp
   assets/wikelo/strata-heatwave.webp
   assets/wikelo/killshot-dominion-reference.webp
   assets/wikelo/r97-crimson-reference.webp
   assets/wikelo/polaris-bit-reference.webp
   assets/wikelo/monde-crimson-reference.svg
7. Also keep the included data and .github folders so automatic roster updates continue working.
8. Commit directly to main with a message such as:
   Update toolkit to v74 and optimize Wikelo artwork
9. Wait for GitHub Pages to deploy.
10. Open the published site and perform a hard refresh:
    Windows/Linux: Ctrl + Shift + R
    macOS: Command + Shift + R
11. If an installed PWA still shows the previous version, close it fully and reopen it twice. In a stubborn browser cache, unregister the old service worker and clear site data.

IMPORTANT
---------
The assets/wikelo folder is required. Uploading only index.html will leave the newly added cards without their local pictures.

ATTRIBUTION
-----------
The toolkit displays source credits on each bundled card. Star Citizen and related game content belong to Cloud Imperium Games. Wiki-derived imagery is attributed in the interface to the Star Citizen Wiki / Wikelo page. The Monde Crimson image is a Celestial Nexus visual reference and is labeled as such.
