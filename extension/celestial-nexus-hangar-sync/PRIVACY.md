# Privacy notice

Celestial Nexus Hangar Sync processes the text content of pledge cards visible on `robertsspaceindustries.com/account/pledges` after the user has signed in to RSI directly.

The extension stores a normalized snapshot locally in browser extension storage. It does not collect or transmit RSI credentials, two-factor authentication codes, cookies, authentication tokens, payment data, or browsing history. The snapshot is shared with the Celestial Nexus GitHub Pages application only when the user requests synchronization from the Fleet Profile module.

No analytics or remote backend are included in this prototype.
