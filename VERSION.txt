Celestial Alliance Nexus ToolKit
Release: v1.2.2
Build: Privacy-minimized Hangar Sync
Published package date: 2026-06-26
Target: GitHub Pages, repository root

Changes in this release:
- Removes raw pledge-card text, pledge identifiers, RSI page title, and source URL from extension storage
- Stores only normalized ship/vehicle names, package name where available, quantity, ownership type, insurance text, and scan time
- Automatically migrates and rewrites existing v1.2.1 extension snapshots into the minimized v1.2.2 format
- Removes the unnecessary activeTab permission; the extension retains only storage plus narrowly scoped content-script hosts
- Purges obsolete pledge identifiers and raw metadata from existing local Fleet Profile entries
- Updates the public privacy notice, store listing, toolkit metadata, service-worker cache, and extension packages

Security model:
- No RSI passwords, 2FA codes, session cookies, authentication tokens, payment data, analytics, advertising SDK, or remote code
- Normalized snapshots remain in browser extension storage until the user connects, exports, or clears them
- No Celestial Nexus cloud backend is included
