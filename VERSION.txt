Celestial Alliance Nexus ToolKit
Release: v1.2.3
Build: Local-Only Toolkit Rollback
Published package date: 2026-06-26
Target: GitHub Pages, repository root

Changes in this release:
- Removes Fleet Profile & Hangar Sync from the toolkit
- Removes the top-toolbar fleet profile control and extension bridge
- Removes automatic pledged-ship ownership detection
- Returns vehicle ownership to the existing manual Already owned / Buy vehicle in-game controls
- Requires no browser extension, external account, cloud service, or backend
- Keeps all Vehicle Manager acquisition planning and shopping routes
- Keeps mining platform recommendations and mining gadgets
- Keeps utility components, mining-module slots, and independent mining/salvage hardpoints
- Keeps Picture Studio premium presets, custom formats, brands, brightness controls, and full-canvas scrolling

Compatibility note:
- This release is based on the stable v1.1.2 application feature set and is numbered v1.2.3 to ensure GitHub Pages and service-worker clients update forward instead of downgrading.
- Old local Fleet Profile data is ignored by this release and is not transmitted anywhere.
