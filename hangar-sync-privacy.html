Celestial Nexus Hangar Sync Privacy Notice

Effective date: 26 June 2026

The extension reads visible pledge information only from the official RSI My Hangar page after the user opens the extension and requests a scan.

Data processed and retained

Before storage, page content is reduced to a normalized snapshot containing detected ship or vehicle names, package name where available, quantity, ownership type, insurance text, and scan time.

Data not retained

The extension does not store raw pledge-card text, pledge identifiers, the RSI page title, source URL, passwords, email credentials, two-factor codes, payment information, session cookies, authentication tokens, or general browsing history.

Storage and sharing

The normalized snapshot is stored in browser extension storage. It is shared only with the Celestial Nexus toolkit after the user requests a connection. This release has no Celestial Nexus server, advertising, analytics, or sale of user information.

Legacy-data migration

When v1.2.2 reads a snapshot created by v1.2.1, it rewrites the snapshot into the minimized format and removes fields no longer used.

User control

Users can export or clear the local snapshot from the extension popup and can remove the extension at any time.

Contact

Support and source information are available through the Celestial Alliance Nexus ToolKit repository.
