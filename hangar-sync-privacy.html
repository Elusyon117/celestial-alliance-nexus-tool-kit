Celestial Nexus Hangar Sync Privacy Notice

Effective date: 26 June 2026

The extension reads pledge-card information only from the official RSI My Hangar page after the user opens the extension and requests a scan.

Data processed

Pledge names, possible included-item names, insurance text, package text, and the scan time may be processed locally to create a fleet snapshot.

Storage and sharing

The snapshot is stored in browser extension storage. It is shared only with the Celestial Nexus toolkit after the user requests a connection. This release has no Celestial Nexus server and does not sell, advertise with, or disclose the snapshot to third parties.

Credentials

The extension does not ask for or intentionally access RSI passwords, email credentials, two-factor codes, payment information, session cookies, or authentication tokens.

User control

Users can export or clear the local snapshot from the extension popup and can remove the extension at any time.

Contact

Support and source information are available through the Celestial Alliance Nexus ToolKit repository.
